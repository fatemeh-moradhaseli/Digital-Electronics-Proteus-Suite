# My Digital Logic Design Projects (Proteus)

This repository contains a collection of my digital logic circuit simulations developed as part of the System Digital Laboratory.
(These projects were part of my earlier academic studies in Electronic in 2019)
## 🛠 What's inside?
I have uploaded several circuit files including:
* Segment Displays: Showing numbers using BCD decoders.
* Binary counters.
* MUX & Logic Gates: Basic combinational circuits and multiplexers.

## 🚀 How to check them
1. Open the .pdsprj files in Proteus.
2. Click the Play button to see the simulation.
3. You can change the logic switches to see how the output changes.

*Note: These are my laboratory assignments and I will update them as I finish more circuits.*
